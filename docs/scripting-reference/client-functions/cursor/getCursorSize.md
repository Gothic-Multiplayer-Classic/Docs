# `function` getCursorSize <font size="4">(client-side)</font>
!!! info "Available since version: 0.3.0"

Returns the cursor size in virtual (screen-scaled) units.

## Declaration
```cpp
int, int getCursorSize()
```

## Parameters
No parameters.
  
## Returns `int, int`
Cursor width and height.

# `function` hexToRgb <font size="4">(shared-side)</font>

Convert hex color to rgb representation.

## Declaration
```cpp
{r, g, b} hexToRgb(string hex)
```

## Parameters
* `string` **hex**: hexadecimal color representation.
  
## Returns `{r, g, b}`
Table containing r, g, b components.

# `function` setPlayerLevel <font size="4">(server-side)</font>

Set a player's experience level.

## Declaration
```cpp
boolean setPlayerLevel(int player_id, int level)
```

## Parameters
* `int` **player_id**: Target player id.
* `int` **level**: New level.
  
## Returns `boolean`
True on success.

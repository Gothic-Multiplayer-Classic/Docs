# `constants` WeaponMode <font size="4">(shared-side)</font>

| Name         | Description                          |
| :----------: | :----------------------------------- |
| `WEAPONMODE_NONE` | No weapon mode. |
| `WEAPONMODE_FIST` | Fist weapon mode. |
| `WEAPONMODE_DAG` | Dagger weapon mode. |
| `WEAPONMODE_1HS` | One-handed sword weapon mode. |
| `WEAPONMODE_2HS` | Two-handed sword weapon mode. |
| `WEAPONMODE_BOW` | Bow weapon mode. |
| `WEAPONMODE_CBOW` | Crossbow weapon mode. |
| `WEAPONMODE_MAG` | Magic weapon mode. |
| `WEAPONMODE_MAX` | Maximum weapon mode value / sentinel. |

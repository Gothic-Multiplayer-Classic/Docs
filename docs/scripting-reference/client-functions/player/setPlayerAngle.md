# `function` setPlayerAngle <font size="4">(client-side)</font>

Set a player's facing angle (radians) in world space.

## Declaration
```cpp
void setPlayerAngle(int player_id, float angle, boolean|nil interpolate)
```

## Parameters
* `int` **player_id**: Target player id.
* `float` **angle**: Angle in radians.
* `boolean|nil` **interpolate**: Optional interpolation flag.

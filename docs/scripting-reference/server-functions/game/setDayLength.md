# `function` setDayLength <font size="4">(server-side)</font>
!!! info "Available since version: 0.3.0"

Set the duration of a full in-game day in milliseconds.

## Declaration
```cpp
void setDayLength(float miliseconds)
```

## Parameters
* `float` **miliseconds**: Day length in milliseconds (min 10000 ms).

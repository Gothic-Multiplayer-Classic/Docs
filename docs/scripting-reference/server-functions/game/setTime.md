# `function` setTime <font size="4">(server-side)</font>

Set the server time (hour, minute, optional day offset).

## Declaration
```cpp
boolean setTime(int hour, int min, int day)
```

## Parameters
* `int` **hour**: Hour (0-23).
* `int` **min**: Minute (0-59).
* `int` **day**: Optional day offset.
  
## Returns `boolean`
True on success.

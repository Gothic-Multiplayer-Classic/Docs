# `function` LOG_WARN <font size="4">(shared-side)</font>
!!! info "Available since version: 0.3.0"

Logs a message with WARN severity.

Warning messages indicate potential problems or unusual situations that
do not immediately stop execution.

## Declaration
```cpp
void LOG_WARN(string text)
```

## Parameters
* `string` **text**: The message text, may contain format specifiers.

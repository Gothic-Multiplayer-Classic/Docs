# `event` onPlayerWhisper <font size="4">(server-side)</font>

Triggered when a player whispers to another player.

## Parameters
```c++
int from_id, int to_id, string text
```

* `int` **from_id**: Sender player id.
* `int` **to_id**: Receiver player id.
* `string` **text**: Whisper text.

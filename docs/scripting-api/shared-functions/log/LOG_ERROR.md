# `function` LOG_ERROR <font size="4">(shared-side)</font>

Logs a message with ERROR severity.

Error messages indicate serious problems that prevent normal operation
or cause a feature to fail.

## Declaration
```cpp
void LOG_ERROR(string text)
```

## Parameters
* `string` **text**: The message text, may contain format specifiers.

# `function` getPlayerHealth <font size="4">(server-side)</font>

Get a player's current health or nil if unavailable.

## Declaration
```cpp
int|nil getPlayerHealth(int player_id)
```

## Parameters
* `int` **player_id**: Target player id.
  
## Returns `int|nil`
Health value or nil.

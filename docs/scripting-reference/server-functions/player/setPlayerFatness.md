# `function` setPlayerFatness <font size="4">(server-side)</font>

Set a player's model fatness.

## Declaration
```cpp
boolean setPlayerFatness(int player_id, float fatness)
```

## Parameters
* `int` **player_id**: Target player id.
* `float` **fatness**: Fatness value.
  
## Returns `boolean`
True on success.

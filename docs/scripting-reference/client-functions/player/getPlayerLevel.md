# `function` getPlayerLevel <font size="4">(client-side)</font>
!!! info "Available since version: 0.3.0"

Get a player's level or 0 if unavailable.

## Declaration
```cpp
int|nil getPlayerLevel(int player_id)
```

## Parameters
* `int` **player_id**: Target player id.
  
## Returns `int|nil`
Level or nil.

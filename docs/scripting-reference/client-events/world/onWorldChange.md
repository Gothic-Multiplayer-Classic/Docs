# `event` onWorldChange <font size="4">(client-side)</font>
!!! info "Available since version: 0.3.0"

Triggered when the client requests a world change.

## Parameters
```c++
void onWorldChange(string world, string waypoint)
```

* `string` **world**: World filename.
* `string` **waypoint**: Waypoint name used for teleport.

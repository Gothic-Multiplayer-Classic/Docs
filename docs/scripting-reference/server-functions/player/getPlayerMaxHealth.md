# `function` getPlayerMaxHealth <font size="4">(server-side)</font>

Get a player's maximum health or nil if unavailable.

## Declaration
```cpp
int|nil getPlayerMaxHealth(int player_id)
```

## Parameters
* `int` **player_id**: Target player id.
  
## Returns `int|nil`
Max health or nil.

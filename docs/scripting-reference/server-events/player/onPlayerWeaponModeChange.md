# `event` onPlayerWeaponModeChange <font size="4">(server-side)</font>
!!! info "Available since version: 0.3.0"

Triggered when a player's weapon mode changes.

## Parameters
```c++
void onPlayerWeaponModeChange(int player_id, int old_mode, int new_mode)
```

* `int` **player_id**: Player id.
* `int` **old_mode**: Previous weapon mode.
* `int` **new_mode**: New weapon mode.

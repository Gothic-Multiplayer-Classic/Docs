# `function` KeyPressed <font size="4">(client-side)</font>

The function is used to check whether the specified keyboard key is pressed.

## Declaration
```cpp
bool KeyPressed(int key)
```

## Parameters
* `int` **key**: The key code to check. For more information about key codes, see [Key Constants](../../client-constants/Key.md).
  
## Returns `bool`
True if the key is currently pressed, false otherwise.

# `function` setCursorSize <font size="4">(client-side)</font>

Sets the cursor size in virtual (screen-scaled) units.

## Declaration
```cpp
void setCursorSize(int width, int height)
```

## Parameters
* `int` **width**: Cursor width.
* `int` **height**: Cursor height.

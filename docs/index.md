## Introduction
Gothic Multiplayer Classic (**GMPC**) is a free multiplayer modification for popular RPG _"Gothic II: Night of the Raven_". Originally founded in 2011, but revisions to the ancient code started in 2022.

The project is developed and maintained by **GMPC Team** and is [open-source](https://github.com/skejt23/GothicMultiplayer).

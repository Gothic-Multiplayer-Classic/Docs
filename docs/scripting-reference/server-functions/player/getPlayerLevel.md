# `function` getPlayerLevel <font size="4">(server-side)</font>

Get a player's level or nil if unavailable.

```cpp
int|nil getPlayerLevel(int player_id)
```

## Parameters
* `int` **player_id**: Target player id.
  
## Returns `int|nil`
Level or nil.

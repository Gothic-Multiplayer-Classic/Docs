# `function` getPlayerAngle <font size="4">(client-side)</font>

Get a player's facing angle (radians) in world space.

## Declaration
```cpp
number|nil getPlayerAngle(int player_id)
```

## Parameters
* `int` **player_id**: Target player id.
  
## Returns `number|nil`
Angle in radians or nil.

# `function` getCursorPosition <font size="4">(client-side)</font>
!!! info "Available since version: 0.3.0"

Returns the cursor position in virtual (screen-scaled) coordinates.

## Declaration
```cpp
int, int getCursorPosition()
```

## Parameters
No parameters.
  
## Returns `int, int`
X and Y position.

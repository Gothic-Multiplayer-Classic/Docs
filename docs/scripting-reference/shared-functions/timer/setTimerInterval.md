# `function` setTimerInterval <font size="4">(shared-side)</font>
!!! info "Available since version: 0.3.0"

Sets the interval (in milliseconds) of an existing timer.

## Declaration
```cpp
void setTimerInterval(int timer_id, int interval)
```

## Parameters
* `int` **timer_id**: Timer ID returned by setTimer.
* `int` **interval**: New interval in milliseconds.

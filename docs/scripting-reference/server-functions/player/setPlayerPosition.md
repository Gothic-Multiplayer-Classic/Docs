# `function` setPlayerPosition <font size="4">(server-side)</font>

Set a player's world position.

## Declaration
```cpp
boolean setPlayerPosition(int player_id, int x, int y, int z)
```

## Parameters
* `int` **player_id**: Target player id.
* `int` **x**: X coordinate.
* `int` **y**: Y coordinate.
* `int` **z**: Z coordinate.
  
## Returns `boolean`
True on success.

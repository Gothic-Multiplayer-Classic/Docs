# `function` getPlayerColor <font size="4">(server-side)</font>

Get a player's name color as a table {r, g, b} or nil if unavailable.

## Declaration
```cpp
{r, g, b}|nil getPlayerColor(int player_id)
```

## Parameters
* `int` **player_id**: Target player id.
  
## Returns `{r, g, b}|nil`
RGB color or nil.

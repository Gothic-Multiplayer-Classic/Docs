# `function` setPlayerColor <font size="4">(server-side)</font>

Set a player's name color (RGB 0-255).

## Declaration
```cpp
boolean setPlayerColor(int player_id, int r, int g, int b)
```

## Parameters
* `int` **player_id**: Target player id.
* `int` **r**: Red (0-255).
* `int` **g**: Green (0-255).
* `int` **b**: Blue (0-255).
  
## Returns `boolean`
True on success.

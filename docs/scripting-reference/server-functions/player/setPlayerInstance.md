# `function` setPlayerInstance <font size="4">(server-side)</font>

Set a player's instance name.

## Declaration
```cpp
boolean setPlayerInstance(int player_id, string instance)
```

## Parameters
* `int` **player_id**: Target player id.
* `string` **instance**: Instance name.
  
## Returns `boolean`
True on success.

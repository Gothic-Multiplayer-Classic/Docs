# `event` onInventorySlotChange <font size="4">(client-side)</font>
!!! info "Available since version: 0.3.0"

Triggered when the selected inventory slot changes.

## Parameters
```c++
void onInventorySlotChange(int from, int to)
```

* `int` **from**: Previous slot number.
* `int` **to**: Current slot number.

# `function` getPlayerMana <font size="4">(server-side)</font>

Get a player's current mana or nil if unavailable.

## Declaration
```cpp
int|nil getPlayerMana(int player_id)
```

## Parameters
* `int` **player_id**: Target player id.
  
## Returns `int|nil`
Mana value or nil.

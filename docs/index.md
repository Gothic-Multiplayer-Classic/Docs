# Home

**Welcome to Gothic Multiplayer Docs**

## Introduction
Gothic Multiplayer Classic (**GMPC**) is a free multiplayer modification for popular RPG "_Gothic II: Night of the Raven_".

Originally founded in 2011, revisions to the original code started in 2022.

---

The project is developed and maintained by **GMPC Team** and is [open-source](https://github.com/skejt23/GothicMultiplayer).

You can contact us on the official [Discord server](https://discord.gg/8BCrFf8qEJ).

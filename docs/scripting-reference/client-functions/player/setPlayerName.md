# `function` setPlayerName <font size="4">(client-side)</font>

Set a player's display name.

## Declaration
```cpp
boolean setPlayerName(int player_id, string name)
```

## Parameters
* `int` **player_id**: Target player id.
* `string` **name**: New player name.
  
## Returns `boolean`
True on success.

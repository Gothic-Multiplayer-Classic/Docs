# `function` getPlayerWorld <font size="4">(server-side)</font>

Get the current world name for a player or nil if player missing.

## Declaration
```cpp
string|nil getPlayerWorld(int player_id)
```

## Parameters
* `int` **player_id**: Target player id.
  
## Returns `string|nil`
World name or nil.

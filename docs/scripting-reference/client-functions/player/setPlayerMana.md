# `function` setPlayerMana <font size="4">(client-side)</font>
!!! info "Available since version: 0.3.0"

Set a player's current mana (clamped to [0,max]).

## Declaration
```cpp
void setPlayerMana(int player_id, int mana)
```

## Parameters
* `int` **player_id**: Target player id.
* `int` **mana**: Mana value.

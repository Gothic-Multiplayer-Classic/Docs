# `function` setPlayerMaxMana <font size="4">(server-side)</font>

Set a player's maximum mana.

## Declaration
```cpp
boolean setPlayerMaxMana(int player_id, int max_mana)
```

## Parameters
* `int` **player_id**: Target player id.
* `int` **max_mana**: New maximum mana.
  
## Returns `boolean`
True on success.

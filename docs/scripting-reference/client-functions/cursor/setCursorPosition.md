# `function` setCursorPosition <font size="4">(client-side)</font>
!!! info "Available since version: 0.3.0"

Sets the cursor position in virtual (screen-scaled) coordinates.

## Declaration
```cpp
void setCursorPosition(int x, int y)
```

## Parameters
* `int` **x**: X position.
* `int` **y**: Y position.

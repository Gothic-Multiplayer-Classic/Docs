# `function` getPlayerAngle <font size="4">(server-side)</font>

Get a player's facing angle in radians or nil if unavailable.

## Declaration
```cpp
int|nil getPlayerAngle(int player_id)
```

## Parameters
* `int` **player_id**: Target player id.
  
## Returns `int|nil`
Angle in radians or nil.

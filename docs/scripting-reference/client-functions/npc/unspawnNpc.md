# `function` unspawnNpc <font size="4">(client-side)</font>

Unspawn (remove) a client NPC from the world without destroying it.

## Declaration
```cpp
boolean unspawnNpc(int npc_id)
```

## Parameters
* `int` **npc_id**: Internal npc id.
  
## Returns `boolean`
True on success.

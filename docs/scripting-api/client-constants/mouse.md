# `constants` Mouse <font size="4">(client-side)</font>

| Name         | Description                          |
| :----------: | :----------------------------------- |
| `MOUSE_DX` | Mouse delta X movement. |
| `MOUSE_DY` | Mouse delta Y movement. |
| `MOUSE_UP` | Mouse movement up. |
| `MOUSE_DOWN` | Mouse movement down. |
| `MOUSE_LEFT` | Mouse left direction. |
| `MOUSE_RIGHT` | Mouse right direction. |
| `MOUSE_WHEELUP` | Mouse wheel scroll up. |
| `MOUSE_WHEELDOWN` | Mouse wheel scroll down. |
| `MOUSE_BUTTONLEFT` | Mouse left button. |
| `MOUSE_BUTTONRIGHT` | Mouse right button. |
| `MOUSE_BUTTONMID` | Mouse middle button. |
| `MOUSE_XBUTTON1` | Mouse extra button 1. |
| `MOUSE_XBUTTON2` | Mouse extra button 2. |
| `MOUSE_XBUTTON3` | Mouse extra button 3. |
| `MOUSE_XBUTTON4` | Mouse extra button 4. |
| `MOUSE_XBUTTON5` | Mouse extra button 5. |

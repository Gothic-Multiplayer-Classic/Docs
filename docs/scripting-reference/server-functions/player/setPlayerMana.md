# `function` setPlayerMana <font size="4">(server-side)</font>
!!! info "Available since version: 0.3.0"

Set a player's current mana.

## Declaration
```cpp
boolean setPlayerMana(int player_id, int mana)
```

## Parameters
* `int` **player_id**: Target player id.
* `int` **mana**: New mana value.
  
## Returns `boolean`
True on success.
